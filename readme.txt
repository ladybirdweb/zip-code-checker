Easily add an ajax-enhanced Zip Code Checker widget to your site's sidebar.

== Description ==

Zip code checker adds a useful widget which you can use to check whether shipping is available on particular area.

= Usage =

Simply go to `Appearance > Widgets` and drag "Zip Code Checker" to the sidebar of your choice. Within the widget are several options you can use for changing the titles or the style displayed to the user.

To use this widget in a template, use [the `the_widget()` function](http://codex.wordpress.org/Function_Reference/the_widget) which WordPress provides.
